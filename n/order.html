순서

직관적으로 생각하는 순서는 정수의 크기비교이다.

⋯≤2≤−1≤0≤1≤2≤⋯\cdots \le 2 \le -1 \le 0 \le 1 \le 2 \le \cdots⋯≤2≤−1≤0≤1≤2≤⋯

순서라는 개념을, 두 원소가 특정 조건을 만족하는 관계로 일반화하여 생각할 수 있다. 조건에 따라 준순서, 부분순서, 전순서 등으로 확장하여 생각할 수 있다.

순서를 표기할 때 ≤\le≤ 기호를 그대로 쓰기도 하고, ≺\prec≺ 기호처럼 살짝 다른 부등호를 써서 정의하기도 한다. 보통 부분순서에 ≺\prec≺, 전순서에 ⪯\preceq⪯ 쓰는 듯.

준순서

어떤 집합 SSS에 대해 이항 관계 ≤\le≤가 준순서(preorder)이려면, ∀a,b,c∈S\forall a, b, c \in S∀a,b,c∈S에 대하여 아래를 만족해야한다.

  • a≤aa \le aa≤a: 반사적
  • a≤b∧b≤c→a≤ca \le b \land b \le c \rightarrow a \le ca≤b∧b≤c→a≤c: 추이적

전순서

어떤 집합 SSS에 대해 이항 관계 ≤\le≤가 전순서(total order)이려면, ∀a,b,c∈S\forall a, b, c \in S∀a,b,c∈S에 대하여 아래를 만족해야한다.

  • a≤aa \le aa≤a: 반사적
  • a≤b∨b≤aa \le b \lor b \le aa≤b∨b≤a: 연결적
  • a≤b∧b≤a→a=ba \le b \land b \le a \rightarrow a = ba≤b∧b≤a→a=b: 반대칭적
  • a≤b∧b≤c→a≤ca \le b \land b \le c \rightarrow a \le ca≤b∧b≤c→a≤c: 추이적

전순서 관계인 집합의 원소는 그 크기를 비교하여 일렬로 늘여 놓을 수 있다. 이 때문에 선형 순서(linear order)라고 부르기도 한다.

부분순서

부분순서(partial order)는 전순서에서 어떤 원소든 비교할 수 있다는 성질이 빠진다.

  • a≤aa \le aa≤a: 반사적
  • a≤b∧b≤a→a=ba \le b \land b \le a \rightarrow a = ba≤b∧b≤a→a=b: 반대칭적
  • a≤b∧b≤c→a≤ca \le b \land b \le c \rightarrow a \le ca≤b∧b≤c→a≤c: 추이적

예시

집합 {x,y,z}\{x, y, z\}{x,y,z}의 부분집합들의 포함관계는 부분순서이다.

  • 모든 원소는 스스로의 부분집합이므로 반사적이다.
  • a⊆b∧b⊆aa \subseteq b \land b \subseteq aa⊆b∧b⊆a인 관계는 a=ba = ba=b일 때 뿐이므로, 반대칭적이다.
  • ∅⊆{x}⊆{x,y}⊆{x,y,z}\emptyset \subseteq \{x\} \subseteq \{x, y\} \subseteq \{x,y,z\}∅⊆{x}⊆{x,y}⊆{x,y,z}인 것 처럼, 추이성도 성립한다.
  • {y}⊈{x,z}\{ y \} \not\subseteq \{x, z\}{y}⊆{x,z}인 것처럼, 모든 관계에서 순서가 성립하는 것이 아니므로 전순서는 아니다.

관계의 성질

아래의 정의는 ∀a,b∧S≠∅\forall a, b \land S \neq \emptyset∀a,b∧S=∅에 대해서이다.

  • R={(a,b)∈A×A∣a∣b}R=\{(a, b) \in A \times A \mid a | b\}R={(a,b)∈A×A∣a∣b}
  • a∣ba|ba∣b는 (a,b)∈R(a, b) \in R(a,b)∈R과 동일하다.

반사적 관계

모든 원소에 대해 스스로와의 관계가 성립한다면 반사적(reflexive)이다.

a∣aa|aa∣a
  • 정수에서 모든 수는 자기 자신과 같고(===), 같거나 작고(≤\le≤), 같거나 크다(≥\ge≥).
  • 집합에서 모든 집합은 자기 자신을 부분집합(⊂\subset⊂)으로 삼는다.

정수의 ≠\ne=, >\gt>, <\lt<는 반사적이지 않다. 이를 비반사적(irreflexive)이라고 한다.

대칭 관계와 반대칭 관계

a∣ba|ba∣b가 성립할 때 그 대칭적 관계, 즉 b∣ab|ab∣a도 성립하면 대칭적(symmetric)이다.

a∣b⇒b∣aa|b \Rightarrow b|aa∣b⇒b∣a

두 항이 같을 때에만 대칭 관계가 성립하면 반대칭적(antisymmetric)이다.

a∣b∧b∣a⇒a=ba|b \land b|a \Rightarrow a = ba∣b∧b∣a⇒a=b

아예 대칭 관계가 성립하지 않으면 비대칭적(asymmetric)이다.

a∣b⇒¬(b∣a)a|b \Rightarrow \lnot (b|a)a∣b⇒¬(b∣a)

추이적 관계

두 관계로 새로운 관계를 유추할 수 있다면 추이적(transitive)이라고 한다.

a∣b∨b∣c⇒a∣ca|b \lor b|c \Rightarrow a|ca∣b∨b∣c⇒a∣c
  • 정수 a, b, c가 있을 때, a<ba < ba<b이고 b<cb < cb<c이면 a<ca < ca<c이다.
    • ∀a,b,c∈R,a<b∧b<c⇒a<c\forall a, b, c \in \mathbb{R}, a < b \land b < c \Rightarrow a < c∀a,b,c∈R,a<b∧b<c⇒a<c
  • 집합 A,B,CA, B, CA,B,C에 대하여, A⊆B∧B⊆C⇒A⊆CA \subseteq B \land B \subseteq C \Rightarrow A \subseteq CA⊆B∧B⊆C⇒A⊆C

연결성?

종류가 여럿 있는데 용어 통일이 안 된 모양새이다. 집합의 전체 원소에 대해 관계를 만족하는 것을 연결되어있다고 한다.

  • 연결되어있다(connected) : a≠b⇒(a∣b∨b∣a)a \neq b \Rightarrow (a|b \lor b|a)a=b⇒(a∣b∨b∣a)
  • 혹은 a∣b∨b∣a∨a=ba|b \lor b|a \lor a = ba∣b∨b∣a∨a=b
  • 강하게 연결되어있다: a∣b∨b∣aa|b \lor b|aa∣b∨b∣a
