수학적 귀납법

수학적 귀납법은 축소 논리이다

n=kn=kn=k일때를 이용해서 n=k+1n=k+1n=k+1로 확장하지 말고, n=k+1n=k+1n=k+1를 이용해서 n=kn=kn=k를 증명하라.

  • 김일희 박사, 수학적 귀납법 99.99% "엉터리"로 배웠습니다. (opens in a new tab).
  • 김일희 박사, 어디서도 보기 힘든 절묘한 수학적 귀납법의 활용 (귀납가설을 다섯번 활용?) (opens in a new tab).

트리로 증명

명제: 트리의의 점 개수가 nnn이면 선 개수는 n−1n-1n−1이다.

  • 트리가 2≤n2 \le n2≤n면 단말이 존재한다.
  • 트리에서 단말을 없애도 여전히 트리이다.

위를 이용하면:

  1. 명제가 참이라고 가정함
  2. n=1n=1n=1일 때, 선의 개수는 0=n−10 = n-10=n−1로 참.
  3. n=k+1n = k + 1n=k+1인 트리가 있다면, 선의 개수는 kkk여야 함.
  4. 위의 트리에서 단말을 없앤 트리는 n=kn=kn=k이고, 가정에 의해 선의 개수는 k−1k-1k−1임.
  5. 단말을 없애면 선도 없어지니 n=k+1⇒kn=k+1 \Rightarrow kn=k+1⇒k가 맞음.

n까지의 합으로 증명

명제: 1+2+⋯+n=n(n+1)21 + 2 + \cdots + n=\frac{n(n+1)}{2}1+2+⋯+n=2n(n+1)​

  1. 명제가 참이라고 가정함.
  2. n=1n=1n=1일 때, 1=221=\frac{2}{2}1=22​이므로 참.
  3. n=k+1n=k+1n=k+1이라고 하면, 1+⋯+k+(k+1)=(k+1)(k+2)21+\cdots+k+(k+1)=\frac{(k+1)(k+2)}{2}1+⋯+k+(k+1)=2(k+1)(k+2)​이어야 함.
  4. 가설에 의해 (1+⋯+k)+(k+1)=k(k+1)2+(k+1)(1+\cdots+k)+(k+1)=\frac{k(k+1)}{2} + (k+1)(1+⋯+k)+(k+1)=2k(k+1)​+(k+1)
  5. 우변의 식을 정리하면 (k+1)(k+2)2\frac{(k+1)(k+2)}{2}2(k+1)(k+2)​이므로 모순이 없으니 명제가 참임.
